# React Internship Portfolio Project

## Overview

This project was developed as part of a React Internship Task.

The application demonstrates:

* Multi-page React application
* React Router navigation
* Dynamic routing
* API integration using Axios
* Todo application with Local Storage
* Reusable React components

## Features

### Task 1: Portfolio Website

* Home Page
* About Page
* Projects Page
* Dynamic Project Details Page
* Contact Information
* Skills and Certifications

### Task 2: API Project

* Fetches user data from JSONPlaceholder API
* Displays user information
* Uses Axios and React Hooks

### Task 3: Todo Application

* Add Tasks
* Delete Tasks
* Local Storage Persistence
* React Hooks

## Technologies Used

* React.js
* React Router DOM
* Axios
* JavaScript
* HTML5
* CSS3
* Vite

## Installation

```bash
npm install
npm run dev
```
